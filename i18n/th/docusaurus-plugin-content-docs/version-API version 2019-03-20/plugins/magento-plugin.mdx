## การติดตั้ง

ใช้ [Composer](/getcomposer.org/) เพื่อดำเนินการติดตั้ง

ให้รันคำสั่งต่อไปนี้ในไดเรกทอรีเดียวกับการติดตั้ง Magento ของคุณ:

```
composer require omise/omise-magento
```
หากคุณเพิ่มโมดูลลงในการติดตั้ง Magento ที่มีอยู่แล้ว คุณอาจต้องดำเนินการคำสั่งดังต่อไปนี้หลังจากที่คุณใช้ Composer แล้ว:
```
php bin/magento module:enable Omise_Payment --clear-static-content
php bin/magento setup:upgrade
php bin/magento setup:di:compile
## การตั้งค่า

เชื่อมต่อบัญชี Omise ของคุณและเปิดใช้งานวิธีการชำระเงินที่คุณต้องการ  
เพื่อให้ลูกค้าของคุณสามารถทำการชำระเงินผ่าน Omise ในร้านค้าของคุณ

ในส่วนของผู้ดูแลระบบ Magento ของคุณ:

1. ไปที่ `Stores` > `Configuration` > `Sales` > `Payment Methods`
2. ตัวเลือกการตั้งค่าจะแสดงในส่วน **Omise**
3. เพิ่ม [รหัสทดสอบ](https://sso-idp.omise.co/realms/engagement/protocol/openid-connect/auth?client_id=dashboard&redirect_uri=https%3A%2F%2Fdashboard.omise.co%2Fv2&state=6ec07f63-85db-4d93-be13-bb75ca934bee&response_mode=fragment&response_type=code&scope=openid&nonce=d13c716a-29c6-4199-8650-4b1de413cbe5&ui_locales=en&code_challenge=4Doxs1FG_nf7vkAZxWvECjIlQ9r97Q0T5FA9Iijzx_M&code_challenge_method=S256) และ [รหัสใช้งานจริง](https://sso-idp.omise.co/realms/engagement/protocol/openid-connect/auth?client_id=dashboard&redirect_uri=https%3A%2F%2Fdashboard.omise.co%2Fv2&state=4c173f49-d4b4-48ba-95f4-13b70a21eaaa&response_mode=fragment&response_type=code&scope=openid&nonce=52d7109e-5bb9-4739-8ce0-67aaaa0d057a&ui_locales=en&code_challenge=NY8CciI7hPbFrW9G0IXNf_tIkQ8KODdO-VKUOEa4HS0&code_challenge_method=S256) แล้วเปิดใช้งานโมดูล

![configuration](/img/configuration.png)

> หมายเหตุ: หากการตั้งค่า `Sandbox` ถูกตั้งค่าเป็น `Yes` จะใช้ **รหัสทดสอบ**  
> หากการตั้งค่า `Sandbox` ถูกตั้งค่าเป็น `No` จะใช้ **รหัสใช้งานจริง**

## เปิดใช้งาน 3DS

หากคุณต้องการประมวลผลการชำระเงินแบบ 3DS (*3-D Secure*)  
[กรุณาติดต่อฝ่ายสนับสนุน](mailto:support@omise.co?subject=Feature%20Request%3A%20Magento) เพื่อเปิดใช้งาน 3DS ในบัญชีของคุณ

> หากทีมงาน Omise เปิดใช้งาน 3DS ในบัญชีของคุณแล้ว แต่ปลั๊กอินยังไม่ได้เปิดใช้งานในร้าน  
> อาจทำให้เกิดปัญหาในการประมวลผลการชำระเงิน  
> คุณจะได้รับการแจ้งเตือนพร้อมข้อความว่า:  
> "3d secure is requested, but return URI is not set"

## อัพเกรดไปยังฟอร์มที่ปลอดภัยใหม่

ในการอัพเกรดไปยังฟอร์มที่ปลอดภัยใหม่ใน Magento ให้ทำตามขั้นตอนต่อไปนี้:

ใช้ **Composer** เพื่ออัพเกรด Magento

รันคำสั่งต่อไปนี้ในไดเรกทอรีเดียวกับการติดตั้ง Magento ของคุณ

```
composer update omise/omise-magento

```
หลังจากนั้นให้รันคำสั่งต่อไปนี้เพื่อเปิดใช้งาน Omise:


```
php bin/magento module:enable Omise_Payment --clear-static-content
php bin/magento setup:upgrade
php bin/magento setup:di:compile
```
